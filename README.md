# Chonk Break

Take breaks seriously — but make them **chonky**.

**Chonk Break** is a lightweight desktop break timer that helps you balance work and rest. Set your work and break durations, and when it’s time to rest, a **Chonk appears on top of everything**, overlaying your screen with a countdown so you *actually take a break*.

![Demo GIF](demo.gif)

## Features

- Set **work / break durations**
- Break time triggers a **fullscreen Chonk overlay**
- Visible **countdown timer**
- Auto-dismiss when break ends
- **System tray integration**
  - Left click: open Control Panel
  - Right click: auto start toggle / open data folder / exit app
- Manual control from Control Panel (mode control & skip state)
- Create and customize your own **Chonk scenes**
- Simple, distraction-free design focused on enforcing healthy breaks

## How to Use

1. Download the latest release for your platform
2. Run the application
3. Left-click the tray icon to open the Control Panel
4. Set and save:
   - Work duration
   - Break duration
5. (Optional) Configure your Chonk scene
6. Let Chonk Break run automatically, or control sessions manually anytime

## Notes

- Break overlay is always on top and gently interrupts your workflow with a friendly Chonk
- Apps remain usable, but Chonk encourages you to step away and rest
- Designed to make breaks feel less like a rule, and more like a moment to pause
- Best for deep work sessions where you tend to forget to rest
